Impact des Jeux Vidéo sur les Performances Académiques

Étude statistique complète sur 175 étudiants, analysant la corrélation entre temps de jeu et résultats scolaires. Conception d'une base de données relationnelle, pipeline ETL complet, interface web déployée sur AWS.

175 Étudiants analysés
−2.6 pts Impact notes (>9h/j)
5+ Tests statistiques
269 Répondants questionnaire

Méthodologie

  • Base relationnelle normalisée (MCD, ETL)
  • Tests Pearson, Spearman, Chi², Entropie
  • Interface filtrage interactif Angular + Flask
  • Hébergement AWS S3 front vitrine

Résultats clés

  • Jouer en anglais → meilleure note en anglais (p≈0)
  • Motivation = facteur déterminant confirmé (p≈0)
  • Temps de jeu seul : corrélation nulle avec les notes
  • Âge moyen des joueurs : 20.19 ans vs 21.38 non-joueurs
